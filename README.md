# Teste 2SOW

Teste realizado para a vaga de front-end developer na 2SOW.

- [Minha experiência](https://github.com/matheuslanduci/vaga-2sow-teste#minha-experiência)
- [Instruções para instalação](https://github.com/matheuslanduci/vaga-2sow-teste#instruções)

## Minha experiência

Estava muito confortável com alguns requisitos do teste: React/Typescript, no entanto tive que aprender algumas novas libs: SemanticUI (simples e fácil), React-Toastify (muito simples) e Testing Library React (complexo e um pouco difícil)

### Sobre os componentes

Apesar de conhecer o SemanticUI, eu não sabia que havia uma biblioteca integrada com o React. Já o Toastify facilita muito qualquer apresentação de mensagem para o cliente - seja de erro, aviso ou sucesso. Apesar de ter sido a primeira vez, com certeza irei utilizar esses componentes em outros projetos.

### Sobre os testes

A parte mais difícil dos testes foi entender a sintaxe do Jest. Depois, criar um caso de uso para teste foi um pouco chato, pois é minha primeira vez com testes e não tinha muito conhecimento sobre isso. No final, consegui criar alguns testes para os componentes da aplicação.

## Instruções

Primeiro, clone para uma pasta:

```
git clone https://github.com/matheuslanduci/vaga-2sow-teste.git app
```

Depois, navegue para a pasta e instale as dependências:

```
cd app
yarn install
```

Depois de instalar as dependências, inicie o servidor de dados:

```
yarn server
```

Agora você pode iniciar o servidor:

```
yarn start
```

Testes unitários disponíveis com:

```
yarn test
```
